# Free Whiteboard Stack Research Notes

This project implementation follows a CPU-first and self-hosted strategy.

## Selected stack

1. Python + Pillow for reliable CPU frame rendering.
2. FFmpeg for MP4 export and audio muxing.
3. Scriptimate as optional open-source CLI SVG animation tool.
4. Piper as optional local realistic TTS voice-over tool.

## Why this stack

1. Fully free/open-source components.
2. Runs on CPU and works in headless CI.
3. Supports daily batch generation through command line.
4. Keeps legal risk low with explicit local asset control.
